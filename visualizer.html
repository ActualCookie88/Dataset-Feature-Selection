Forward selection peak: Features {6, 15} at 95.8% accuracy — up from a baseline of 74.8% using all 16 features. Adding a 3rd feature drops accuracy to 94%, showing that irrelevant features actively degrade nearest-neighbor classification. Both algorithms converge on the same optimal subset for this dataset.
All accuracy values are measured by 1-nearest-neighbor leave-one-out cross-validation on the provided datasets.
Distances are precomputed per-feature before search. Early exit applied when accumulated distance exceeds current best.
